## Como Criar uma Nova Skill

1. Criar pasta em `skills/[nome-da-skill]/`
2. Criar `SKILL.md` com: O que faz, Quando usar, Passo a passo, Dependências, Output
3. (Opcional) Adicionar `examples/` e `scripts/`
4. Atualizar este INDEX.md
5. Se for rotina, adicionar também em `rotinas/INDEX.md`
6. Commit com prefixo `feat: skill [nome]`
