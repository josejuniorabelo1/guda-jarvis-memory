## Resumo Executivo

O Módulo 1 ensina a construir o "Segundo Cérebro" — um repo GitHub que centraliza todo conhecimento da empresa para qualquer agente acessar.

### Pilares
1. **Cérebro** — GitHub como fonte de verdade (estrutura: empresa/areas/agentes/seguranca)
2. **Skills** — Automações em linguagem natural (receitas reutilizáveis)
3. **Crons + Heartbeats** — Agenda operacional + cérebro estratégico
4. **Segurança** — 3 camadas (servidor, agente, processo)

### Regras Fundamentais
- Toda pasta precisa de MAPA.md (índice)
- Skills > Prompts (permanentes, consistentes, evoluem)
- Começar simples, escalar conforme necessidade
- Credenciais nunca no código
- A lógica importa mais que a ferramenta
